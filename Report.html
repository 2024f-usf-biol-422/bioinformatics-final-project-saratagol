Analysis of SARS-CoV-2 Sequencing Data from Experimentally Infected Hamsters and Ferrets

Sara Tagol

02 December, 2024

Background and Overview

This is a report on SARS-CoV-2, including some variant analysis (Koyama et al., 2020). This report focuses on the analysis of SARS-CoV-2 sequencing data alongside tabular data, including a variant analysis and data summary. The primary goal is to investigate patterns in viral variants and correlate them with metadata such as geographic location.

Key objectives include: 1. Summarizing sequencing quality. 2. Identifying frequent variants in the VCF files. 3. Correlating variant frequency with metadata from the SraRunTable.

Sequencing Data Analysis

Sequencing data were processed using bwa for alignment and bcftools for variant calling. The resulting VCF files were analyzed for variant frequencies and annotated with metadata.

Tabular Data Analysis

The SraRunTable.csv was analyzed in R to summarize sample metadata and investigate patterns by region and sequencing platform.

All analyses were conducted using R scripts integrated into this RMarkdown file, ensuring reproducibility.
